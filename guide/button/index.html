Navigation buttons

Navigation buttons are used at the top of each page and are designed to be the same height as a primary button with the btn-small class applied.

They are not designed to be used elsewhere on the site.

Tags

Tags sit on the Insights landing page, on individual Insight article pages and on the tag listing pages. They are also used as a call-to-action link style in some Insight call-to-action panels.

FAQ Buttons

FAQ buttons can be used to trigger FAQs (obviously), but are also used on the process page to allow the user to jump further down the page.

They require a span tag to be nested inside the parent element. By default, they take up the full width of their parent element.
