Quotations

Quotations reuse heading styles, but with the addition of quotation marks around the text being quoted

Blockquotes can be used as asides in Insight articles or to illustrate a point in the main content.

Sometimes, quotations go on for more than one paragraph. When this happens, only the last paragraph should have a closing quotation mark.

Like this one, for example.

Text can take any size of heading text

If you're using background colours and padding utility classes, make sure to match the padding class with the size of the heading.

You do not need to add a background colour or padding utility class to blockquote elements, if you don't want to.
